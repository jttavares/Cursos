Introdução

A web existe desde o início dos anos 1990 e, de lá para cá, a forma como os sites são apresentados mudou drasticamente, com uma clara evolução das técnicas de desenvolvimento de aplicações Web. Isso é justificado em parte por causa da relevância que a plataforma Web assumiu, principalmente nos últimos anos, “tomando” o espaço de aplicações desktop, por exemplo. Com essa evolução, começaram a surgir ferramentas e frameworks que visam acelerar o processo de aplicações Web.

Uma das ferramentas que podemos utilizar para acelerar o desenvolvimento de interfaces para aplicações Web é justamente o Bootstrap. Certamente a utilização deste no desenvolvimento de uma página web irá acelerar e facilitar substancialmente o desenvolvimento de uma página com uma aparência atrativa e com um design adaptativo e responsivo.
