O que é bootstrap?

O Bootstrap é um dos mais famosos e utilizados frameworks CSS da atualidade. Isso quer dizer que ele foi concebido para facilitar a parte de estilização das páginas que vierem a utiliza

O Bootstrap foi criado pela equipe do Twitter inicialmente, mas depois teve seu código aberto e hoje possui uma vasta comunidade empenhada em manter o framework evoluindo. No Brasil, um de seus maiores sponsors é o portal Globo.com. A equipe do portal Globo.com inclusive criou uma versão própria do Bootstrap que é utilizada nos portais vinculados, como o Globo.tv Se você deseja conhecer um pouco sobre o projeto em cima do Bootstrap do time do portal Globo.com, você pode acessar o link: http://globocom.github.io/bootstrap/

Porém, é importante ressaltar que neste curso nós utilizaremos a versão “oficial”.

Nós aqui da TreinaWeb também utilizamos o Bootstrap em nosso site público e também em nosso ambiente educacional – o painel do aluno e o AVA.

O Bootstrap trabalha em cima de algumas premissas:

  • Facilidade de uso e implementação: o Bootstrap é muito intuitivo (muito mesmo!) e simples de entender, o que certamente facilita a sua implementação;
  • Concebido para ser mobile-first: o Boostrap é pensado para funcionar como um framework CSS mobile-first. Isso quer dizer que, o utilizando corretamente, você consegue desenvolver páginas com layouts adaptativos e responsivos para quaisquer dispositivos existentes atualmente com quase nenhum esforço.

O conceito mobile-first é relacionado com uma cultura de desenvolvimento de interfaces. De maneira tradicional, nós desenvolvemos os layouts de nossas páginas pensando que elas serão inicialmente acessadas através de computadores, para depois fazermos adaptações para que as páginas sejam renderizadas sem problemas nos demais dispositivos, principalmente os celulares e smartphones. Porém, hoje temos um cenário muito peculiar: em 2016, a quantidade de acessos à internet através de celulares e smartphones ultrapassou a quantidade de acessos através de computadores e notebooks. Para você ter uma ideia, em cerca de 80% dos municípios do Brasil, o principal meio de acesso à internet são os celulares e smartphones, ao passo de que o acesso tradicional através de computadores e notebooks vem caindo ano após ano. Por isso, hoje, a cultura de desenvolvimento se inverteu: é muito comum as páginas web serem desenvolvidas primeiro pensando-se em dispositivos mobile para depois serem adaptadas para que sejam executadas nos demais dispositivos, como os computadores. Hoje, primeiro se pensa em como a página será renderizada nos dispositivos mobile, para depois as adaptações serem feitas para os demais dispositivos. Daí vem o conceito de mobile-first. O Bootstrap é excelente nesse sentido, pois ele já é concebido para trabalhar naturalmente com este novo conceito.

Apesar de a grande vocação do Bootstrap ser a sua parte de CSS, hoje ele também já abrange componentes HTML e JavaScript, caminhando para ser um grande e poderoso framework front-end de maneira geral. Porém, atualmente, seu aspecto voltado a CSS continua sendo o grande chamariz e diferencial.
