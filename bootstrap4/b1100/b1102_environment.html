Estrutura básica

Containers

Elementos html que contem outros elementos dentro dele como body, div, etc.

Todo container precisa possuir uma classe container ou container-fluid.

  • container: possui medida fixa;
  • container-fuid: adapta-se à resolução do dispositivo.
